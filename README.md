# sim-based-inf-clin
This repository accompanies the manuscript in preparation "Distinct patterns of slow repolarisation underlie abnormal ECGs in hypertrophic cardiomyopathy"

## Overview
Implementation of a simulation-based framework to infer ventricular activation and repolarisation times from a biventricular mesh and 12-lead ECG signal.

## Environment
Python 3.12.6 (64-bit Windows, MSC v.1940)
